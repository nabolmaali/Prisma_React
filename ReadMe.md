# PrismaReact

PrismaReact is a frontend application built with **React and TypeScript**
that consumes the **Prisma_API** backend (which is implemented by .NET Core API - under repository named Prisma_Repository).

This project is designed to demonstrate frontend architecture,
component design, and API integration patterns.
